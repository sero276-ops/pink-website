О моём сайте

Этот сайт создан для курсового проекта на Coursera. В нём используются современные CSS свойства:

  • Flex и Grid для макета
  • Hover эффекты на изображениях и ссылках
  • Псевдоклассы nth-child
  • Адаптивный дизайн с использованием meta viewport
